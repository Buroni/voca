Languages

Currently supported languages are English, Dutch, French, German, Italian and Spanish.

Adding Languages

Supported languages are limited by available sentence data, and the language support provided by the Pattern library, which is used for word inflection. Future support might be possible by using additional language processing libraries, or providing the option to lookup sentences in new languages without the inflection feature.
